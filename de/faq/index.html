FAQ

Wie installiere ich die Feinstaub-App auf meinem Gerät?

Navigiere Dein Android-Gerät zum Google PlayStore und suche in der Suchleiste nach 'Feinstaub-App'. Tippe auf den obersten Listeneintrag und anschließend auf 'Installieren'. Die App wird nun auf Deinem Gerät installiert. Tippe auf 'Öffnen' um die App zu starten.

Hinweis: Die App wird im Moment nur für Android-Geräte unterstützt. Falls Du kein Android-Gerät besitzt, kannst Du auch unsere Web-App ausprobieren.

Wie verknüpfe ich einen Feinstaubsensor?

Es gibt zwei Möglichkeiten einen Feinstaubsensor in der App zu verknüpfen:

Sensor auf der Karte zu den Favoriten hinzufügen:
Navigiere auf die Startseite der App auf den Reiter 'Alle Sensoren'. Suche auf der Karte den Sensor, den Du zu den Favoriten hinzufügen möchtest. Tippe auf den Sensor und auf 'Favorit hinzufügen'. Anschließend kanst Du dem verknüpften Sensor noch einen benutzerdefinierten Namen zuweisen. Sobald der Sensor verknüpft wurde, ist er auf der Seite 'Meine Favoriten' zu finden.

Eigenen Sensor verknüpfen:
Navigiere auf den Reiter 'Meine Sensoren' und tippe auf die '+'-Schaltfläche unten rechts. Fülle die Daten des Sensors wahrheitsgemäß aus. Wir bitten darum, den genauen Standort des Sensors anzugeben. Tippe anschließend auf das kleine Häkchen oben rechts. Nun sollte der Sensor auf der Seite 'Meine Sensoren' zu finden sein.

Wie bearbeite ich einen verknüpften Sensor?

Wechsle auf die Seite, auf der Dein verknüpfter Sensor angezeigt wird und tippe auf die drei Punkte rechts auf dem Sensor-Item (Bei eigenen Sensoren auf der Favoriten-Seite: Navigiere bitte auf die Seite 'Meine Sensoren'). Tippe im Menü auf 'Sensor bearbeiten'. Sobald Du mit der Bearbeitung fertig bist, tippe rechts oben auf das kleine Häkchen.

Wie entferne ich die Verknüpfung eines Sensors?

Wechsle auf die Seite, auf der Dein verknüpfter Sensor angezeigt wird und tippe auf die drei Punkte rechts auf dem Sensor-Item (Bei eigenen Sensoren auf der Favoriten-Seite: Navigiere bitte auf die Seite 'Meine Sensoren'). Tippe im Menü auf 'Sensor enternen'. Bestätige den Dialog mit 'Sensor entfernen'.

Wie zeige ich die Messdaten für einen Sensor an?

Hier gibt es zwei Möglichkeiten. Entweder Sie erstellen eine Verknüpfung (siehe oben) und tippen darauf oder Sie navigieren zur Karte, tippen einen Sensor an und wählen 'Messwerte zeigen'.

Wie setzte ich das Häkchen in der Sensor-Konfiguration?

Um das Häkchen in der Sensor-Konfiguration zu setzen, musstt Du dich mit dem Sensor im selben Netzwerk (z.B.: im selben WLAN-Netz) befinden. Gib die IP-Adresse des Sensors in die Adressleiste des Browsers ein. Es sollten mehrere Buttons angezeigt werden. Tippe auf 'Konfiguration' und auf den Reiter 'APIs'. Aktiviere dort die Option 'Senden an Feinstaub-App' und tippe auf 'Speichern und neu starten'. Es kann bis zu 10 Minuten dauern, bis Messdaten auf unserem Server ankommen. Bitte hab Geduld, bis Du den Sensor in der App verknüpfen kannst.
