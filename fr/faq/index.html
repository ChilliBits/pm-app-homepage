FAQ

Comment puis-je installer l'App Matière Particulaire sur mon appareil?

Rendez sur votre appareil Android au Google PlayStore et cherchez 'App Matière Particulaire'. Appuyez sur le premier résultat de recherche, puis sur 'Installer'. L'application est ensuite téléchargée et installée sur votre appareil. Appuyez sur 'Ouvrir' pour lancer l'application.

Information: Actuellement, l'application ne prend en charge que les appareils Android. Si vous n'avez pas d'appareil Android, vous pouvez essayer notre application de web qui a les mêmes fonctionnalités.

Comment lier un capteur pm?

Il existe deux façons différentes de lier un capteur de pm dans l'application:

Ajouter un capteur à la carte
Accédez à l'onglet 'Tous les capteurs' et cherchez un capteur, que vous souhaitez ajouter à vos favoris. Appuyez sur le marqueur et sur 'Mettre en favori'. Enfin, vous pouvez choisir un nom personnalisé pour le capteur. Dès que le capteur est lié, il peut être vu sur le tab 'Mes favoris'.

Link propre capteur
Naviguez sur l'onglet 'Mes capteurs' et appuyez sur la '+' pointe de menton en bas à droite. Remplissez les vraies données du capteur. Nous vous demandons, d'en préciser l'emplacement précis. Appuyez sur la petite icône terminée dans le coin supérieur droit. Maitenante, le capteur doit être trouvé sur l'onglet 'Mes capteurs'.

Comment éditer un capteur existant?

Passez à l'onglet, où votre capteur s'affiche et appuyez sur les trois points à droite de l'élément du capteur (Pour vos propres capteurs, vous devez vous rendre sur la page 'Mes capteurs'). Appuyez sur 'Modifier le capteur'. Dès que vous avez terminé de réactualiser le capteur, appuyez sur l'icône 'Terminé' dans le coin supérieur droit.

Comment supprimer un lien existant d'un capteur?

Passez à l'onglet, où votre capteur s'affiche et appuyez sur les trois points à droite de l'élément du capteur (Pour vos propres capteurs, vous devez vous rendre sur la page 'Mes capteurs'). Appuyez sur 'Supprimer le capteur'. Confirmez la boîte de dialogue en appuyant sur 'Supprimer le capteur'.

Comment puis-je afficher les données de mesure d'un capteur spécifique?

Ici, il y a deux façons. Soit vous créez un lien (voir ci-dessus) et appuyez dessus ou accédez à la carte, appuyez sur un capteur et sélectionnez 'Afficher les valeurs'.

Comment activer le 'tick' dans la configuration du capteur?

Pour activer le 'tick' dans la configuration du capteur, vous devez être dans le même réseau local avec le capteur (par exemple le même réseau WiFi local). Saisissez l'adresse IP du capteur dans la barre d'adresse de votre navigateur, s'il vous plaît. Il devrait y avoir plusieurs boutons. Appuyez sur 'Configuration' et accédez à l'onglet 'API'. Activez l'option 'Envoyer à Feinstaub-App' et appuyez sur 'Sauvegarder et redémarrer'. Cela peut durer jusqu'à 10 minutes pour recevoir des données du capteur. Soyez patient jusqu'à ce que vous puissiez lier le capteur dans l'application, s'il vous plaît.
